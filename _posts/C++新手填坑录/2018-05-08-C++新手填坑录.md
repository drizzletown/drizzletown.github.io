## 错误1：变量初始化问题
- 代码示例：计算1+2+3+...+100。
```
int val;
for(int i=0; i<100; i++)
	val += i;
cout<<val<<endl;
```
- 执行结果

随机值，不是期望的5050。
- 原因分析及解决方案

变量val定义时未进行初始化工作，故其初始值是一个随机值。将1、2、...、100累加到val中结果依然是一个不确定的值。解决办法是给变量val设定初值，
```
double val = 0;
```
